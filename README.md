# LupusLink

## Purpose
LupusLink is a supportive web platform designed to help people living with lupus, their families, and caregivers access trusted information, community support, and daily wellness tools. The project aims to make lupus care feel more approachable, organized, and empowering.

## Key Features
- Informative landing page about lupus and community support
- Separate sign-up and sign-in pages for new and returning users
- Personalized dashboard for wellness tracking
- Medication tracking with checkable tasks and progress updates
- Symptom summary with visual severity indicators
- Upcoming appointment reminders and scheduling support
- Daily mood check-in
- Water intake tracker
- Rest reminder with sun/UV wellness guidance
- Profile page for personal details, notifications, and account management

## Technologies Used
- HTML5 for page structure
- CSS3 for styling, layout, and responsive design
- LocalStorage for saving dashboard preferences such as mood and water intake

## Web Pages
- Home - Main landing page
- Login - New user registration page
